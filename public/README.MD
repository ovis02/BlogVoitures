# Blog Automobile

## Description

Ce projet est un blog dédié aux voitures légendaires, mettant en avant six modèles emblématiques de l'histoire automobile. Les utilisateurs peuvent voter pour leur voiture préférée et laisser des commentaires.

## Fonctionnalités

- Une fonctionnalité permet aux utilisateurs d’allumer ou d’éteindre les phares des voitures via une simple interaction, ajoutant une dimension interactive.
- Affichage des voitures avec des descriptions détaillées.
- Système de vote permettant aux utilisateurs de choisir leur voiture préférée.
- Formulaire de commentaires pour recueillir les avis des visiteurs.
- Interface d'administration pour gérer les commentaires.

## Technologies Utilisées

**HTML** : Structure de base du site web.

- **CSS** : Stylisation du contenu pour un design moderne et responsive.
- **Bootstrap** : Cadre CSS utilisé pour rendre le site responsive.
- **JavaScript** : Interaction dynamique, y compris la gestion des animations, des phares, et des formulaires.
- **PHP** : Gestion des formulaires et traitement des votes en back-end.
- **MySQL** : Base de données utilisée pour stocker les informations sur les voitures, les votes, et les commentaires.
